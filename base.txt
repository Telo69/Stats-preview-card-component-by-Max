index.php
Этот код представляет собой базовую структуру HTML-документа со встроенными элементами разметки и стилей. Внутри тега  определены различные настройки и свойства страницы, такие как кодировка символов, указание на CSS-стили и другие метатеги. 


Внутри тега  определен основной контент страницы, который представляет собой два элемента: секцию и блок с фоновым изображением. Внутри секции находится заголовок, надпись, список из трех элементов, каждый из которых содержит заголовок и абзац. 


Второй блок представляет собой фоновое изображение, на котором через CSS задаются наложенные цветовые слои с прозрачностью.




style.css
Этот код импортирует шрифты «Inter» и «Lexend Deca» из Google Fonts и устанавливает несколько переменных CSS для цветов и фоновых изображений. Он также включает медиа-запросы для настройки макета и размеров шрифта для небольших экранов. Основные стили CSS задают цвет фона для тела, режим отображения для основного раздела и стили для различных разделов и элементов в основном разделе.
